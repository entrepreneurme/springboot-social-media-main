Person API

Create your own person with a POST:

https://predjeskovic.com/socialmedia/people

Requested Body: {

"firstName:" "firstName",

"lastName:" "lastName",

"username:" "username",

"email:" "email" }

Find the person your created by typing the username from your creation with a GET:

https://predjeskovic.com/socialmedia/people/username

To see all the People with a GET:

https://predjeskovic.com/socialmedia/people
